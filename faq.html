FAQs

  1. IS INTERNET CONNECTIVITY REQUIRED FOR 24x7 FOR THE APPLICATION?
    No, it is not required. Once you download the latest update sent by school, you can disconnect internet.

  2. WHAT DO YOU MEAN BY SECURITY CODE?
    Security code is a unique identification for your school, you will get this from School or from AcadePro team.

  3. PARENTS CAN COMMUNICATE WITH SCHOOL STAFF REGARDING QUERY?
    No, currently this facility is not available for parents all information is view only. We have Appointment option through that you can fix appointment with school.

  4. IS SCHOOL STUDY MATERIAL AVAILABLE ON ACADEPRO?
    No, currently there is no any separate provision for uploading the study material for students, but school can use Notice option to provide reference URL for any attached document to help student.

  5. WHAT ABOUT THE NON-ANDROID/iOS MOBILE USER, HOW THEY CAN UPDATED?
    School can send the simple text SMS to Non-Android mobile users using web application

  6. CAN I SEE ALL THE DETAILS OF BOTH CHILDS IN ONE MOBILE, WHICH ARE STUDYING IN SAME SCHOOL?
    Yes, Siblings are automatically managed in single login if the same mobile given for both students otherwise you can use the Add Student functionality

  7. I AM TRYING TO SYNC BUT IT IS SHOWING MESSAGE "SYNC FAILED"
    There are two possibilities- i) if user id & password is not matching with system ii) If Network is weak

  8. IS IT NECESSARY TO SIGN UP FOR LOGIN INTO APP?
    No. You can chose to directly input your User id i.e. mobile no. & Password

  9. I DO NOT HAVE ANDROID/iOS PHONE OR OTHER SMART PHONE, SHALL I'LL ABLE TO USE WEB LOGIN OR ANY OTHER MEDIUM TO VIEW CHILD INFORMATION?
    You can't access the App on the web.

  10. CAN I CHANGE THE GIVEN USERNAME AND PASSWORD?
    You can not change the username & Password from the App. You will need to contact your School/AcadePro for the same.

  11. IS IT POSSIBLE TO DOWNLOAD IMAGES IN MY MOBILE FROM MEDIA SECTION?
    No, all media uploaded y school is view only you can't download the same.

  12. HOW I CAN GIVE FEEDBACK TO SCHOOL OR REGISTER ANY COMPLAIN?
    You can call us from the call support number that is given on the dashboard of the app.

  13. CAN WE SEND SMS TO PARENTS USING ACADEPRO APP?
    Yes, You can Send SMS through Teacher App to specific student & to all student.

  14. CAN TEACHERS OR STAFF APPLY FOR LEAVE USING ACADEPRO APP?
    No, currently leave application facility is not available on Teachers app. For leave application you need to login with web application with your User name and password and apply for leave.

  15. CAN WE SEND HOME WORK ONLY TO SINGLE STUDENT OR COUPLE OF STUDENTS USING ACADEPRO APP?
    You can push home work to class and divisions only. You cannot push home work only for selected students using app

  16. CAN I CHANGE MY PASSWORD USING ACADEPRO APP?
    No, You cannot change the password using the App. To change the password you need to contact your School admin or login on web portal and go to the change password option.

  17. CAN I MANAGE THE EXTRA OR SUBSTITUTE PERIOD USING ACADEPRO APP?
    Yes, after login with app just click on Other Class option and select respective Board, Class and Division.

  18. CAN I DO ONE TO ONE COMMUNICATION WITH PARENT AND TEACHER USING ACADEPRO APP?
    No, for security reasons this facility is not available. But Parent can fix appointment with teacher.

  19. CAN WE USE ACADEPRO APP OFFLINE?
    Yes, you can use Teacher's App online and offline as well, but internet connectivity is compulsory to Sync the application and to send the data to the server.

  20. I DO NOT HAVE PLAY STORE LOGIN, WHAT TO DO?
    Please create your gmail account & then login into play store. If you already have gmail account then register it on play store while login.
